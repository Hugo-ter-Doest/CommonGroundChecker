# Public icons

Place static icon assets in this folder.

Usage in Next.js:

- `/icons/my-icon.svg`
- `/icons/my-icon.png`

Examples:

```tsx
<img src="/icons/my-icon.svg" alt="My icon" />
```

```tsx
import Image from "next/image";

<Image src="/icons/my-icon.svg" alt="My icon" width={24} height={24} />
```
